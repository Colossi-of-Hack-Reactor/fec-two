### Setup Instructions
1. Install yarn globally via npm:

  ```
  $npm i -g yarn
  ```

2. Install all dependencies via yarn:
  ```
  $yarn install
  ```

3. Make a copy of example.env, rename it to .env, and change the values of the properties in the file as desired/necessary.
